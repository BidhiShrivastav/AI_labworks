# RNN Encoder–Decoder with Luong Dot Attention

This project implements a **Sequence-to-Sequence (Seq2Seq) RNN Encoder–Decoder** with the **Luong Dot Attention** mechanism using **PyTorch** for neural machine translation.

## Features

- RNN-based Encoder–Decoder architecture
- Luong Dot Attention mechanism
- Teacher Forcing during training
- Sequence-to-Sequence translation
- Implemented using PyTorch

## Requirements

- Python 3.x
- PyTorch
- NumPy
- Matplotlib





## Results

The Luong Dot Attention mechanism improves translation performance by allowing the decoder to focus on the most relevant encoder outputs, resulting in better sequence prediction compared to a basic Encoder–Decoder model.
